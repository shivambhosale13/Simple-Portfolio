Profile Card UI – HTML & CSS

A clean and modern Profile Card UI built using pure HTML and CSS.
It includes a round profile image, social links, action buttons, and analytics counters.

Features

Responsive and minimal UI

Circular profile picture with border

Social media icon buttons

Subscribe and Message buttons

Analytics section (Likes, Comments, Shares)

Google Fonts (Poppins)

Boxicons for icons

Project Structure
/
├── Profile Card.html   # Main HTML file
├── style.css           # Stylesheet
└── me.jpg              # Profile photo (you must add this file)

How to Use

Upload all project files to your GitHub repository

Ensure Profile Card.html and style.css are in the same folder

Add your profile image as me.jpg

Open Profile Card.html in a browser to preview the UI

Technologies Used

HTML5

CSS3

Google Fonts

Boxicons Icons
